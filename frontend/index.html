Tipos de alerta
🔴 Caro: seu preco mais de 10% acima do menor concorrente
🟡 Atencao: 5-10% acima do menor concorrente
💚 Oportunidade: voce e o mais barato — pode subir a margem

Configuracao Z-API
1. Acesse z-api.io, crie uma conta e conecte seu WhatsApp via QR Code.
2. Abra config/config.json e preencha: 
"whatsapp": {
  "enabled": true,
  "provider": "zapi",
  "instance_id": "SEU_INSTANCE_ID",
  "token": "SEU_TOKEN",
  "numbers": ["5533327129120"]
}
3. Inicie o servidor:
python backend/server.py
